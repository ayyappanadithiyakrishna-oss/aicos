# AICOS Terminal

The Next.js trading terminal and AI Investment Committee for AICOS — the
AI Investment Committee Operating System. Dark, Bloomberg-grade UI in the
Scale design language; a live committee of six specialist agents that debate
a ticker before capital moves.

## Stack

- Next.js 16 (App Router) · TypeScript · Tailwind v4 · Framer Motion
- TradingView Advanced Chart embed
- `@anthropic-ai/sdk` — committee reasoning (`claude-opus-4-8`)

## Run

```bash
npm install
cp .env.example .env.local   # optional: add ANTHROPIC_API_KEY to go live
npm run dev                  # http://localhost:3000  (terminal at /terminal)
```

## Live data

| Surface | Source | Notes |
|---|---|---|
| Equity quotes | Yahoo Finance (`/api/quote`) | Real, no key. Polled every 15s; "LIVE" badge when fresh, falls back to the seeded book offline. |
| Committee debate | Anthropic (`/api/committee`, SSE) | Generated by `claude-opus-4-8` when `ANTHROPIC_API_KEY` is set; deterministic simulation otherwise. The transcript header shows **Live AI** vs **Sim**. |
| Chart | TradingView widget | Live, client-side. |
| Portfolio | Seeded demo book | Brokerage (Robinhood) integration lives server-side in the Python orchestrator; the web app shows the demo account. |

## Going live with the committee

1. `cp .env.example .env.local`
2. Set `ANTHROPIC_API_KEY=sk-ant-...`
3. Restart `npm run dev`. The committee now streams Claude-generated debate; the
   header badge flips from **Sim** to **Live AI**. No code change required —
   `/api/committee` swaps the source and keeps the same streamed shape.

## Routes

- `/` — marketing / hero
- `/terminal` — the trading terminal + committee
- `/api/quote?symbols=NVDA,AAPL` — live quotes JSON
- `/api/committee?ticker=NVDA` — committee meeting (Server-Sent Events)
